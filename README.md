# Social Links Profile

## Overview

This project is a solution to the Social Links Profile challenge.
The goal is to recreate the provided design as accurately as possible while ensuring a smooth and accessible user experience.

The project focuses on clean layout, proper styling, and interactive states for better usability.


## The Challenge

The challenge was to build out a social links profile page that closely matches the given design.

You are free to use any tools or technologies of your choice to complete the challenge and practice your skills.
